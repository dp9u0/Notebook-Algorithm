* 有序表查找:    二分查找  
* 分块查找:     块内无序，块之间有序；可以先二分查找定位到块，然后再到`块`中顺序查找  
* 动态查找:     二叉排序树，AVL树，B- ，B+(这里之所以叫 `动态查找表`，是因为表结构是查找的过程中动态生成的)
