# 线性数据结构

线性数据结构是最基本数据结构,其元素是顺序或者链式排列的.这也是线性数据结构的两种结构类型: 顺序结构和链式结构. 顺序结构中,元素是在内存中相邻排列,例如数组.而链式结构中,元素在内存中不一定是相邻的,但是通过指针链接起来,逻辑上是线性的.
